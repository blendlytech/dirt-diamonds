# **Gritty Events & Hustle Mechanics**

## **Event Dispatcher**

Runs on a background polling loop checking the Entity_Flags and Players tables.

Event JSON Structure:

Prerequisites (e.g., Funds < 500 AND Recklessness > 75).

Probability Weight (chance of firing).

Branching Choices.

Choices must write immediate consequences (deduct funds) AND write hidden flags to Entity_Flags to trigger cascading events years later.

Hustle State Machines
Hustles operate as isolated interactive nodes.

Narcotics: 3-tier system (Inventory Drop -> Profit/Toxicity Cut -> Territory Control vs Factions).

Texas Hold'em: Direct mathematical card simulation evaluating pot odds.

Legal Work: Time-skip abstractions that heavily drain Energy and Fitness needs for minimal payout.
