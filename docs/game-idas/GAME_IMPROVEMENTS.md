# Game Improvements - Dirt & Diamonds

## Overview

This document contains the game improvement ideas for the game Dirt & Diamonds.  

## Story Line

- Add more story lines
- Make the story line more interesting
- Make the story line more engaging
- Make the story line more immersive
- Make the story line more interactive
- Make the story line more replayable

## Game UI

- Calendar should be moved into cellphone
- Move game time into cellphone
- Actual time of day should be visible on the phone screen and should change like a real clock. ( Maybe 1 minute in game = 1 second in real life )
- Time should be able to pause, speed up, and slow down.
- Add save button into cellphone
- Game autosaves after every message or event.
- Add options into cellphone
- Make UI more immersive ( maybe different ui for different locations )
- Add more UI animations
- Cellphone should be more interactive
- Cellphone types need to be sylized ( different models for different tiers )
- Add text message features to cellphone
- Remove pop up notifications when using items
- Add sound effects for all UI elements

## Gameplay

- Add more activites to do
- Add more things to collect
- Add a tutorial on how to play the game
- Add a section on how to play the game in cellphone at the very start of the game to teach players the basics
- Add more things to do in the game
- Add more things to collect in the game
- Add more things to buy in the game
- Add more things to upgrade in the game
- Add more things to discover in the game
- Add more things to earn in the game
- Add more things to lose in the game

## Hustles

- Make all jobs interactive ( instead of just clicking a button to get money )
- Texas Holdem should be an actual full mini-game of Texas Holdem with different locations and stakes based on player progress.

## Criminal Side Hustles

- Drug dealing, should be series of progressive mini games with stages.
- Fencing should be a series of progressive mini games with stages.
- Make robberies more interactive ( instead of just clicking a button to rob )
